# Security Policy

## Supported Versions

All versions will have supported security. <br>
However, beta releases will not be promised to meet security standards.
| Version | Supported          |
| ------- | ------------------ |
| 1.0.0   | :white_check_mark: |
| 1.0.1 -beta  | :x:                |


## Reporting a Vulnerability

You can report a vulnerability at hello@syop200.com. <br>
I will reach out as soon as I can after a security issue is brought to my attention.
